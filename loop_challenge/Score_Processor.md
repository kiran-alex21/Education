# Score Processor

## Main Task

Create a program that can:
    - Take a list of scores
    - Tell the user:
        - How many scores where entered
        - The average score
        - The highest score
        - How many people got the highest score

Trial list: 52,16,7,26,72,35,5,56,34,72

## Extention 1

The list now include the names of the score holder
Alter your output to display the:
    - Top 5 scores
    - Top 5 score holder names

## Extension 2

The client now wants the program to run with a GUI.
Create a list of requirements for the GUI
Make a basic starter file (like the ones in this task)
Use Tkinter to build your GUI
